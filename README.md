# Stories of Unais

A full-stack story publishing platform.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Vite
- **Backend**: Node.js, Express, JSON-DB (File-based)
- **Auth**: JWT

## Getting Started

### 1. Start the Backend Server
The backend runs on port 3001 and handles data persistence.

```bash
# Install backend dependencies (if running locally)
# npm install express cors jsonwebtoken

# Run the server
node server/index.js
```

### 2. Start the Frontend
The frontend runs on Vite (usually port 5173) and connects to the backend at http://localhost:3001.

```bash
# Install frontend dependencies
npm install

# Run the frontend
npm run dev
```
